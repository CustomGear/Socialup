How to use:
Click inside the white box → Select All (⌘A) → Copy (⌘C) → open Gmail / Apple Mail / Outlook signature settings → paste (⌘V).

Gmail tip: Settings → See all settings → General → Signature → Create new → paste.
Apple Mail tip: Mail → Settings → Signatures → + → paste.
